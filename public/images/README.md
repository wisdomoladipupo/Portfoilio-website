# Project Images

Please add your project screenshots and profile picture to this folder:

## Required Images:
- `profile.jpg` - Your profile picture
- `school-cbt-dashboard.jpg` - School CBT Admin Portal Dashboard screenshot
- `school-cbt-manage-users.jpg` - School CBT Manage Users screenshot  
- `runtech-roadmaps.jpg` - RunTech Roadmaps page screenshot

## Image Specifications:
- Recommended size: 1200x800px for project screenshots
- Profile picture: 400x400px (square)
- Format: JPG, PNG, or WebP
- Keep file sizes under 500KB for faster loading

Once you add the images, they will automatically appear in your portfolio!
